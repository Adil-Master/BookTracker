# Трекер книг - программа для трекинга прочитанных книг
## Версия 1.0 | Доступно:
- Добавление книг
- Удаление книг
- Получение средней оценки
- Получение списка книг
- Статистика по авторам
## Как запустить?
1. Установите актуальную версию программы.
2. Распакуйте архив.

|№ |Linux | Windows |
|:---| :---: | :---: |
|1.| Откройте в терминале директорию с распакованной программой | Откройте директорию с распакованной программой в проводнике |
|2.| Запустите программу при помощи комманды _**python3 main.py**_ | Запустите файл main.py двойным кликом|
|-| - | ИЛИ |
|3.| - | Откройте в терминале директорию с распакованной программой  |
|4.| - | Запустите программу при помощи комманды _**python main.py**_  |
